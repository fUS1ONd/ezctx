# Phase 5.5: Validation Strategy — Clipboard Engine (super_clipboard)

## Test Map

| ID | Type | Description | Automated | Wave |
|----|------|-------------|-----------|------|
| CP-01 | manual | Транскрипция 1.5-часового аудио копируется без обрезания на реальном Android-устройстве | No | post-deploy |
| CP-02a | unit | `ClipboardService.copyText` вызывает `clipboard.write` с `DataWriterItem` содержащим plain text | Yes | wave-1 |
| CP-02b | unit | При `SystemClipboard.instance == null` бросается `StateError` | Yes | wave-1 |
| CP-02c | unit | При `PlatformException` от `clipboard.write` — исключение пробрасывается наружу | Yes | wave-1 |
| CP-03 | static | `result_screen.dart` не содержит `Clipboard.setData` (grep возвращает 0) | Yes | wave-1 |
| CP-04 | static | `AndroidManifest.xml` содержит `SuperClipboardDataProvider` (grep возвращает 1+) | Yes | wave-1 |

## Validation Architecture

### Wave 1 (automated, при каждом коммите)

1. Unit-тесты ClipboardService:
   ```
   /opt/flutter/bin/flutter test test/core/services/clipboard_service_test.dart
   ```
   Покрывает: CP-02a, CP-02b, CP-02c

2. Статические проверки:
   ```
   grep -c "Clipboard.setData" lib/ui/screens/result_screen.dart  # должно вернуть 0
   grep -c "SuperClipboardDataProvider" android/app/src/main/AndroidManifest.xml  # должно вернуть 1
   ```
   Покрывает: CP-03, CP-04

### Post-deploy (manual)

CP-01 — выполняет пользователь после установки APK:
- Загрузить аудио длительностью ≥ 1.5 часа
- Запустить транскрипцию
- Нажать «Скопировать»
- Вставить в текстовый редактор (Telegram, Заметки)
- Убедиться, что текст не обрезан на ~21-й минуте
