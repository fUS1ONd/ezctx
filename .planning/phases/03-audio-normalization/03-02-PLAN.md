    Было: сразу определялся `_isChunked = args.isChunked`, затем запускался контроллер.

    ```dart
    @override
    void didChangeDependencies() {
      super.didChangeDependencies();
      if (_file == null) {
        final args = ModalRoute.of(context)?.settings.arguments;

        SelectedAudioFile? file;
        if (args is ProcessingArgs) {
          file = args.file;
        } else if (args is SelectedAudioFile) {
          // Обратная совместимость
          file = args;
        }

        if (file != null) {
          _file = file;
          _startedAt = DateTime.now();
          _ticker = Timer.periodic(const Duration(seconds: 1), (_) {
            if (mounted) {
              setState(() => _elapsed = DateTime.now().difference(_startedAt!));
            }
          });
          // Нормализация запускается после первого кадра
          WidgetsBinding.instance.addPostFrameCallback((_) => _startProcessing());
        } else {
          WidgetsBinding.instance.addPostFrameCallback((_) {
            if (mounted) Navigator.popUntil(context, (r) => r.isFirst);
          });
        }
      }
    }
    ```

    --- НОВЫЙ МЕТОД _startProcessing ---
    Добавить приватный async-метод:

    ```dart
    /// Шаг 1: нормализация → шаг 2: ветвление на chunked / single.
    Future<void> _startProcessing() async {
      if (!mounted) return;

      // Показать «Подготовка аудио…»
      setState(() => _normalizing = true);

      try {
        _normalizedFile = await AudioNormalizationService().normalize(_file!.path);
        _isChunked = _normalizedFile!.durationSeconds > AppConstants.kChunkThresholdSeconds;
      } catch (e) {
        // Ошибка нормализации — показать через TranscriptionController (будет TranscriptionError)
        setState(() => _normalizing = false);
        // Перебросить через контроллер чтобы отобразить в UI
        _controller.emitError('Ошибка подготовки аудио: $e');
        return;
      }

      if (!mounted) return;
      setState(() => _normalizing = false);

      // Конструировать SelectedAudioFile из нормализованного пути для передачи контроллерам
      final normalizedAudioFile = SelectedAudioFile(
        path: _normalizedFile!.path,
        name: _file!.name,
        sizeBytes: File(_normalizedFile!.path).statSync().size,
        extension: 'mp3',
      );

      if (_isChunked) {
        _chunkedController = ChunkedTranscriptionController(
          keyRepository: ApiKeyRepository(SecureStorageServiceImpl()),
          apiService: GroqApiService(),
          chunkingService: AudioChunkingService(),
        );
        _chunkedController!.addListener(_onChunkedStateChange);
        _chunkedController!.start(normalizedAudioFile);
      } else {
        _controller.start(normalizedAudioFile);
      }
    }
    ```

    Примечание: `File` уже импортирован через dart:io в существующих импортах. Если нет —
    добавить `import 'dart:io';`.

    Примечание о `_controller.emitError`: если TranscriptionController не имеет такого метода,
    использовать альтернативу. Проверить наличие метода в TranscriptionController.
    Если метода нет — использовать setState для прямого показа ошибки через локальное поле
    `String? _normalizationError`, добавить его в State и показывать в build(). Не блокировать
    выполнение попыткой вызова несуществующего метода.
