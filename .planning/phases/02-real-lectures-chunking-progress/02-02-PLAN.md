# Plan 02-02: ChunkedTranscriptionController — параллельность + retry + сборка

## Objective

Реализовать `ChunkedTranscriptionController` — ChangeNotifier, управляющий всем пайплайном
для больших файлов: разбивка через `AudioChunkingService`, параллельная отправка с bounded
concurrency (_Semaphore, max 3), автоматический retry на транзиентных ошибках,
сборка финального текста с таймкодами, удаление tmp-чанков.

Запускается параллельно с Plan 02-01 (разные файлы).

`groq_api_service.dart` — добавить метод `transcribeChunk`:
```dart
/// Транскрибация одного чанка из байт. Используется в Phase 2 чанкованием.
/// Бросает [AuthException] / [NetworkException] / [RateLimitException] / [InternalException].
Future<TranscriptionResult> transcribeChunk({
  required List<int> bytes,
  required String filename,
  required String apiKey,
}) async {
  final client = _clientFactory();
  try {
    final uri = Uri.parse(AppConstants.groqApiUrl);
    final request = http.MultipartRequest('POST', uri)
      ..headers['Authorization'] = 'Bearer $apiKey'
      ..fields['model'] = AppConstants.groqDefaultModel
      ..fields['response_format'] = AppConstants.groqResponseFormat
      ..fields['timestamp_granularities[]'] = AppConstants.groqTimestampGranularity
      ..fields['timestamp_granularities[1]'] = 'segment'  // нужен для сборки таймкодов по чанкам
      ..files.add(http.MultipartFile.fromBytes('file', bytes, filename: filename));

    final streamed = await client.send(request).timeout(
      const Duration(minutes: 5),
      onTimeout: () {
        client.close();
        throw const NetworkException('Превышено время ожидания ответа от Groq');
      },
    );
    final response = await http.Response.fromStream(streamed);

    if (response.statusCode == 200) {
      try {
        final json = jsonDecode(response.body) as Map<String, dynamic>;
        return TranscriptionResult.fromJson(json);
      } catch (_) {
        throw const InternalException('Не удалось разобрать ответ Groq');
      }
    }
    if (response.statusCode == 401) throw const AuthException(_authErrorMessage);
    if (response.statusCode == 429) throw const RateLimitException('Превышен лимит запросов Groq');
    // 5xx, 524
    throw const NetworkException(_networkErrorMessage);
  } on SocketException { throw const NetworkException(_networkErrorMessage); }
  on TimeoutException { throw const NetworkException(_networkErrorMessage); }
  on AppException { rethrow; }
  catch (_) { throw const NetworkException(_networkErrorMessage); }
  finally { client.close(); }
}
```

`chunked_transcription_controller.dart`:

Вспомогательный класс `_Semaphore` в начале файла (приватный):
```dart
class _Semaphore {
  _Semaphore(this._maxConcurrent);
  final int _maxConcurrent;
  int _running = 0;
  final _queue = <Completer<void>>[];

  Future<T> run<T>(Future<T> Function() fn) async {
    if (_running >= _maxConcurrent) {
      final waiter = Completer<void>();
      _queue.add(waiter);
      await waiter.future;
    }
    _running++;
    try {
      return await fn();
    } finally {
      _running--;
      if (_queue.isNotEmpty) _queue.removeAt(0).complete();
    }
  }
}
```

Метод `start(SelectedAudioFile file)`:

2. `final keys = await _keys.listKeys()` → если пусто → `ChunkedMissingKey()`
3. `final chunkFiles = await _chunkingService.split(file.path)` → в try/catch → `ChunkedError`
4. Инициализировать `List<ChunkState>` как `[ChunkWaiting(0), ChunkWaiting(1), ...]`
5. `_set(ChunkedProcessing(chunks: ..., completedCount: 0, totalCount: n))`

7. В блоке `finally`: удалить tmp-чанки `for (final f in chunkFiles) { try { await f.delete(); } catch(_){} }`
8. Собрать результат → `_set(ChunkedSuccess(result))`

Вспомогательный `_withRetry<T>` — принимает `chunkIndex` для обновления состояния чанка:
```dart
Future<T> _withRetry<T>(
  Future<T> Function() fn,
  int chunkIndex, {
  int maxRetries = 3,
}) async {
  int attempt = 0;
  Duration delay = const Duration(seconds: 5);
  while (true) {
    try {
      return await fn();
    } catch (e) {
      if (e is AuthException) rethrow; // не ретраить Auth
      if (e is! NetworkException && e is! RateLimitException) rethrow;
      if (++attempt >= maxRetries) rethrow;
      _updateChunkState(chunkIndex, ChunkRetrying(chunkIndex, attempt: attempt));
      await Future.delayed(delay);
      delay *= 2;
    }
  }
}
```

Вызов из `_processChunk(int index, File file, String apiKey)`:
```dart
final result = await _withRetry(
  () => _api.transcribeChunk(bytes: ..., filename: ..., apiKey: apiKey),

```

Сборка финального текста в `_assembleResult(List<TranscriptionResult> results, double chunkDuration)`:
- Для каждого чанка `i`, для каждого segment в `results[i].segments`:
  - `absoluteStart = i * chunkDuration + segment.start`
  - Форматировать как `[HH:MM:SS]`
  - Текст: `[HH:MM:SS] segment.text\n`
- Финальный `TranscriptionResult` — text = собранный текст, language = results[0].language, duration = sum of all durations

Для вычисления `chunkDuration` — использовать `AudioChunkingService.getMetadata().durationSeconds / chunkCount` (если метаданные доступны) либо 1200.0 как константу (CHUNK_DURATION_SECONDS из AudioChunkingService).
